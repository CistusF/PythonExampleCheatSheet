만약 실행시 오류가 발생할 경우
Visual Studio Code일 경우가 99.999%이므로 파이썬 idle를 이용하여 실행해보시길 바랍니다.


open 코드 리스트

w = 파일을 쓰기위해 사용
    이 때 입력받은 파일 이름을 생성하며
    이미 있을 경우 기존 파일을 지우고 새로 생성한다.
r = 파일을 읽기위해 사용
    파일이 존재하지 않을 경우 오류가 발생
a = 파일을 쓰기위해 사용
    만약 파일이 없을 경우 파일을 생성하며
    이미 있을 경우 기존 파일에 내용을 추가하는 구조를 가진다.
r+ = 파일을 읽고 쓰기 위해 사용된다.
rb = 이진파일을 읽기 위해 사용된다. 
    r과 동일하게 생각하면 편하다.
wb = 이진 파일을 쓰기 위해 사용된다.
    w와 동일하게 생각하면 편하다.
